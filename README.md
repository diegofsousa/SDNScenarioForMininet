# SDNScenarioForMininet

Cenário criado para simulação de uma Rede Definida por Software (SDN)

Este modelo é composto por:

1 Controlador (c0);<br>
4 Switches (s1, s2, s3, s4);<br>
6 Hosts (h1, h2, h3, h4, h5, h6, h7, h8).<br><br>
